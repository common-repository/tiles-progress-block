Add a progress bar block to your Gutenberg built website with the Tiles Progress Bar block.

== Description ==

A quick and simple way to add progress bars to your website.

== Installation ==
1. Upload the `tiles-progress-bar` folder to your `/wp-content/plugins/` directory or alternatively upload the tiles-progress-bar.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your local computer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Search for the block within the Gutenberg block editor.

== Frequently Asked Questions ==

= Can this Gutenbeg block be used on my theme? =
Yes! You can use this block on any Gutenberg block-based, full site editing, or universal WordPress theme.
